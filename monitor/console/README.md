Dependences
===========

The PIL module need to be complie with `libjpeg-devel` and `freetype-devel` .



Usage
=====

    Create data directories:

        % mkdir samples out


    In `ipython` shell:

        %run experiment.py
        fetch(code='Z9010', dt=None, num=12, rate=15)
        process(num=12, factor=10, threshold=0)
